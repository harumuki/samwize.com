date: 2016-01-25T16:56:06+08:00

| What              | Shortcut                  |
|:------------------|:--------------------------|
| Run all the test  | ⌘ U                       |
| Run 1 test        | ctrl ⌥ ⌘ U                |
| Re-run last       | ctrl ⌥ ⌘ G                |

Note: Run 1 test will run the test that your cursor is on.
