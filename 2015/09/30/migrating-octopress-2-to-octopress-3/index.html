Migrating Octopress 2 to Octopress 3

Published on 30 Sep 2015



read

Octopress 3.0 is coming. It has been 9 months, but they still have not officially released it.

However, you can use pre-released Octopress 3.0.

This post is a guide to migrating an existing Octopress 2.0 website.

Installing

mkdir mywebiste.com
cd mywebiste.com
touch Gemfile

Edit the Gemfile as follows:

In Octopress 2.0, octopress is a derivative of jekyll with more features.

In Octopress 3.0, octopress is merely a plugin to jekyll. And there are other plugins provided by octopress such as codefence for backtick code blocks.

Run bundle install to install the gem.

Setup a new website

octopress new mywebsite.com

You can now run: 

cd mywebsite.com
jekyll s

Plugins

To activate the jekyll plugins, edit _config.yml and add this line:

gems: [octopress-image-tag, octopress-video-tag, octopress-codeblock, octopress-quote-tag, octopress-codefence]

Migrating Your Posts

Migrating octopress/jekyll is easy, because the source are your markdown files.

Copy all the markdown post in _posts from your old Octopress to this new Octopress.

Themes

Jekyll themes are merely fork of Jekyll with custom html/css/etc.

For this website, I use mediator.

Download/Clone mediator, then copy

  • _config.yml
  • _includes
  • _layouts
  • _sass
  • assets
  • css
  • feed.xml
  • index.html

You will also need the merge the theme's Gemfile to yours.
